ISTRUZIONI INSTALLAZIONE STAZIONE VIDEO

1- Copiare la cartella videoPlayer all'interno del raspberry

2- Eseguire tutte le installazioni necessarie (jdk,mvn....)
	sudo apt install openjdk-17-jdk
	sudo apt install maven

2- Attivare il servizio automatico (guarda systemctl)

3- Attivare il wifi-automatico

4- Disabilitare standby e spegnimento automatici

5- Inserire una USB con dentro un .mp4, .jpj, .json come nella cartella USB_TEMPLATE
